\section{Alphabetic Labels}
Legacy style guides often prefer alphabetic labels for stable review drafts.
The core syntax still uses the usual citation command, so the prose can cite
\cite{knuth84}, combine sources such as \cite{lam94,hoare69}, and revisit
\cite{bentley86} later in the same note.

\section{Editing Workflow}
Because the labels are authored in the bibliography itself, teams can keep the
text stable while the bibliography grows. This makes annotated reading lists
easy to compare across revisions.
